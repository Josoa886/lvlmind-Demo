About our website


Mini projet LevelMind
Notre mini projet est un sous projet de notre projet e-learning LevelMind qui est un vaste
projet de site e-learning. Nous avons créé ce site dans afin d'aider principalement les
collégiens et les lycéens ainsi que les étudiants qui veulent développer leurs connaissances
ou aussi ceux qui veulent s'instruire via une autre forme d'éducation. Mais cela
n'empêche pas ceux qui veulent étendre leurs connaissances d'y accéder
qu'ils soient jeunes ou adultes. Ce site peut remplir le rôle qu'a un
professeur ou enseignant mais entre autre satisfaire
les demandes de l'apprentit.
